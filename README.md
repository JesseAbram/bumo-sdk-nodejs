bumo-sdk
=======

## bumo-sdk  Installation
```
npm install bumo-sdk --save
```

## bumo-sdk  Test
```
npm test
```

## bumo-sdk  Usage

```js
'use strict';

//Create BumoSDK instance
const BumoSDK = require('bumo-sdk');

const sdk = new BumoSDK({
  host: 'seed1.bumotest.io:26002',
});


```
